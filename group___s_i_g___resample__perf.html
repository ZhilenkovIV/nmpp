Подробное описание

Функции для оценки производительности функций фильтрации SIG_Resample()

Функция эквивалентно следующим псевдоинструкциям:

void SIG_Resample***_perf(nm16u15b* pSrcVec, nm16u15b* pDstVec, int nSrcVecSize, nm64s* pKernel);
{
SIG_SetResample***_***(pKernel);
clock_t t0,t1;
t0=clock();
SIG_Resample***(pSrcVec, pDstVec, nSize,pKernel);
t1=clock();
exit(t1-t0);
}
Возвращаемые значения
pKernelУказатель на служебную структуру, содержащую весовые коэффиценты фильтра во внутреннем формате.
Аргументы
pSrcVecВходной вектор.
nSrcVecSizeРазмер входного вектора в элементах.
Возвращаемые значения
pDstVecРезультирующий вектор.
Заметки
Инициализация служебной структуры производится соответсвующей функцией SIG_SetFilter() и SIG_CreateFilter().
