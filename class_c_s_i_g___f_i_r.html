Подробное описание

template<class nmbits_in, class nmbits_out>
class CSIG_FIR< nmbits_in, nmbits_out >

Класс КИХ фильтра

Template Parameters
nmbits_inТип указывающий разрядность входного вектора. Допустимые типы : nm8s,nm16s,nm32s
nmbits_outТип указывающий разрядность выходного вектора. Допустимые типы : nm8s,nm16s,nm32s. Разрядность входного вектора не должна превышать разрядности выходного.
